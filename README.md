Nexmo Notifier
==============

Provides Nexmo integration for Symfony Notifier.

DSN example
-----------

```
// .env file
NEXMO_DSN=nexmo://API_KEY:API_SECRET@default?from=FROM
```

where:
 - `API_KEY` is your Nexmo API key
 - `API_SECRET` is your Nexmo API secret
 - `FROM` is the sender
